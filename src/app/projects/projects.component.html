My Projects

Here is a list of projects I have made and my contributions to the Open-Sourec ♥

Quiz App

This is a simple quiz app made in flutter that take questions from an online API and puts in front of it. It can be run both on Android and IOS

It also shows the difficulty and genre of the questions asked. The questions can be MCQ's or Boolean based.

Technologies Used : Android, Flutter

Arcade Game

This is a simple arcade game made in Javascript. This is an implementation of Frogger game where user has to go through a lane avoiding the beetles.

They score a point once they cross the lane without getting hit. With increasing score, the number of beetles increases.

Technologies Used : HTML5, CSS3, Javascript

Pixel Art Maker

A simple pixel art maker that helps you make your favourite pixel art made simply using Javascript.

In this the user defines the height and the width of the pixel board. After this the user choses the color and paints what ever they want.

Technologies Used : HTML5, CSS3, Javascript

Movie Trailer Website

This is a simple movie trailer website that displays a list of movies made completely using Python3.

In this the user clicks the movie image and the trailer for the same is displayed from Youtube in the website only !

Technologies Used : Python3, HTML5, CSS3

Neighbourhood Map

This is a neighbourhood map made with Javascript and Four-Square API

This gives you the location needed and the places near it.

Technologies Used : Javascript
